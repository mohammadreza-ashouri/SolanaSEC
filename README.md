# SolanaSECs

I have been coding a tool that allows to perform automated security analysis on Rust Solana programs.

## What vulnerabilites SolanaSECs supports now?

-  overUnderFlow
-  missingSigner
-  account_confusions
-  missing_rent

## How to use it?

 Example: python3 solanasecs.py -f tests/data/example.rs
